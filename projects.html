My Projects

Elasto

Elasto is my second official game project. It's a top-down survival themed game where you are supposed to fight the zombies to survive and obtain keys. Those keys are used to either open doors, to discover the map, or to upgrade your power like speed and strength. The project was made in a total of 2 working months, and uploaded to my Itch IO page.

Technologies and Tools Used:

  1. Unity Game Engine: to make the game functions and everything
  2. Aseprite: to draw the different game assest
  3. Visual Studio Code: to code with intellesence
Note that I put this in the beginning because it's my biggest and best project.

Ribbat

Ribbat is my first official game. In this game, you are a rabbit trying to find the end line while spending the lowest time possible, typically a speedrun game. There is also carrots around the map which will, upon catching, decrease the timer by 3s. The game isn't too big yet it took around a full month to finish including the drawing state, that was mainly because I didn't have that experience in the Game-Dev world yet. Make sure to try it also on my Itch IO page.

Technologies and Tools Used:

  1. Unity Game Engine: to make the game functions and everything
  2. Aseprite: to draw the rabbit and tilemaps
  3. Visual Studio Code: to code with intellesence
  4. Itch IO: to get some assets
